02 / Reading

Article Copy Stack

Default long-form rhythm for narrative text, links, bullets and subheads.

Width
720px max copy column
Spacing
3rem section bottom, 1.5rem gutters

More Than One Story Layer

The format works best when each chapter combines a concise analytical point with one visual job. A paragraph can include a source link and a small inline emphasis treatment without becoming a graphic.

When a section introduces multiple constraints, bullets keep the reading path stable:

  • One idea per bullet, written as editorial copy rather than interface instructions.
  • Bullets should not become a replacement for the visual sequence.
  • Use this pattern when the client needs detail but not a new asset.

03 / Static Media

Single Image Widths

Use single images when one finished asset can carry the point. Width controls should be chosen based on detail level, label density and how much the image should interrupt the reading flow.

Variants
Column-width, wide-width, full-page width
Scope
1 final exported image per figure

04 / Sticky Media

Scroll-Swap Sequences

Use when the reader needs to compare states in place. Each specimen below uses dummy frames, explicit step counts and the same progressive swap behavior as the production pattern.

Performance
Lazy dummy frames, reduced-motion support
Controls
Step count, aspect ratio, fade duration, final hold

Three-Step Contained Swap

Best for intro maps, before-during-after geography, or compact data states. The final frame holds longer so the reader can inspect the destination state.

04b / Sticky Media

Fast Panoramic Swap

A two-state wide image sequence for changes that should read quickly and cleanly.

Media

Scope
2 exported frames, fast fade

Fast Panoramic Swap

Use when the image is wide and the state change should feel immediate, such as route closures, port access, or a single annotated overlay.

04c / Sticky Media

Extended Sequence

A longer sticky sequence for process stories where each frame adds a meaningful layer.

Media

Scope

Extended Sequence

Reserve for complex processes where each frame adds a meaningful layer. Five steps is a higher-scope asset request and should be approved early.
